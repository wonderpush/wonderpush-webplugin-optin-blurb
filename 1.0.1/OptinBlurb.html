Class: OptinBlurb

OptinBlurb

new OptinBlurb(WonderPushSDK, options)

WonderPush Web SDK plugin to present the user an opt-in blurb inciting users to subscribe to push notifications.

Parameters:
Name Type Description
WonderPushSDK external:WonderPushPluginSDK

The WonderPush SDK instance provided automatically on instantiation.
options OptinBlurb.Options

The plugin options.

  • Object
Properties:
Name Type Argument Description
message String <optional>

The blurb text.
urlFilters WonderPushPluginSDK.URLFilters | undefined <optional>

URL filters to apply before showing the blurb.
noIcon boolean <optional>

Do not display an icon.
querySelector String <optional>

The blurb will be installed inside nodes matching this query selector.
insertBefore boolean <optional>

When true, the blurb is inserted before any other child of the parent. Defaults to false, meaning the blurb is inserted after all children.
cssPrefix String <optional>

Defaults to 'wonderpush-'.
linkStyle String <optional>

Will be used as the "style" attribute of the link element;
style String <optional>

Will be used as the "style" attribute of the container element;
bellIconColor String <optional>

Color of the bell icon. Defaults to #ff6f61
alignment String <optional>

'left', 'right' or 'center'. Defaults to 'left'.
bellIcon String <optional>

URL of the bell icon. Use a transpared image whose alpha channel will be used to draw the icon, in the color specified by bellIconColor.
horizontalPadding String <optional>

Horizontal padding of blurb. Defaults to 0
verticalPadding String <optional>

Horizontal padding of blurb. Defaults to 0
imgUrl String <optional>

When given, the specified image will be used instead of the bell icon.
multiple boolean <optional>

When true, the blurb will be installed inside each node returned by document.querySelectorAll. Otherwise, the blurb will be installed in the node returned by document.querySelector(). Defaults to false.
